# 技术指标速查表

信号分析 tab 底部的 14 个按钮对应 14 种常用技术指标(由 HQChart 图表库内置)。本文按**"看什么 → 怎么读 → 常见用法"**三段式说明每个,方便边看回测 K 线边对照。

> 术语:K 线 = 蜡烛图的一根(一个时间周期);周期参数默认多为 N=12 或 14 根;**金叉**/**死叉** = 指标两条线向上/向下交叉(常被当作买/卖信号)。

---

## 一、成交量类

### VOL — 成交量 (Volume)
- **看什么**:每根 K 线对应周期内成交了多少手
- **怎么读**:柱子越高代表当时交易越活跃。红柱 = 该周期收阳(涨),绿柱 = 收阴(跌)
- **常见用法**:价格突破新高**伴随**放量 → 趋势可信;无量突破容易假突破

### OBV — 能量潮 (On-Balance Volume)
- **看什么**:累积成交量(阳 K 加,阴 K 减),反映资金净流入
- **怎么读**:OBV 新高 → 资金持续流入;OBV 与价格背离 → 警惕趋势反转
- **常见用法**:配合价格看"量价齐升"还是"量价背离"

---

## 二、动量/强弱类

### RSI — 相对强弱指数 (Relative Strength Index)
- **看什么**:一段时间内涨幅 vs 跌幅的比值,0~100 震荡
- **怎么读**:
  - **> 70 超买**(可能回调)
  - **< 30 超卖**(可能反弹)
  - 50 中性线
- **常见用法**:短线逆势交易的择时参考;趋势强时超买可能持续很久不是卖点

### KDJ — 随机指标 (Stochastic)
- **看什么**:K、D、J 三条线,衡量价格在最近 N 根 K 线高低价区间的相对位置
- **怎么读**:
  - K 上穿 D = **金叉**(买入信号)
  - K 下穿 D = **死叉**(卖出信号)
  - J 是 K、D 的加速信号,更灵敏
  - KD 都 > 80 超买,都 < 20 超卖
- **常见用法**:震荡市高胜率,趋势强时容易过早反向

### MACD — 平滑异同移动平均 (Moving Average Convergence Divergence)
- **看什么**:DIF(快慢均线差)、DEA(DIF 均线)、柱状(两者差)
- **怎么读**:
  - DIF 上穿 DEA 且在零轴上 = 强势买入
  - 柱状由绿转红 = 动能转多
  - 柱状由红转绿 = 动能转空
- **常见用法**:趋势市最经典的择时工具;**背离**(价格新高但柱状没新高)常是反转前兆

### ROC — 变动率 (Rate of Change)
- **看什么**:当前价 相对 N 周期前价的百分比变化
- **怎么读**:ROC 上穿 0 = 上升动能启动;下穿 0 = 转弱
- **常见用法**:量化动量策略的核心因子之一

### CCI — 顺势指标 (Commodity Channel Index)
- **看什么**:当前价偏离"近期典型价平均"的标准化程度,无上下限
- **怎么读**:
  - **> +100 进入多头区**(趋势买点)
  - **< -100 进入空头区**(趋势卖点)
  - ±100 之间震荡市
- **常见用法**:捕捉强势趋势的启动点,比 RSI 更适合趋势市

### BIAS — 乖离率
- **看什么**:当前价与 N 日均线的百分比偏离
- **怎么读**:绝对值越大 → 超买/超卖越严重,均值回归概率越高
- **常见用法**:反趋势交易的过滤条件(如 BIAS > 10% 不追高)

---

## 三、均线类

### HMA — 赫尔移动平均 (Hull Moving Average)
- **看什么**:一种"既快又平滑"的加权均线(Alan Hull 发明)
- **怎么读**:比同周期 SMA/EMA 转向更快,噪音更少
- **常见用法**:趋势跟踪的入场/止损线,适合替代普通均线

### LMA / VMA — 加权 / 可变移动平均
- **LMA**(Linear Weighted Moving Average,线性加权):越近的 K 线权重越大,比普通 SMA 对新价格反应更快
- **VMA**(Variable MA,可变权重):根据波动率自动调整周期,波动大时更平滑、波动小时更敏感
- **常见用法**:用在趋势识别中,两者都是"比 SMA 灵敏,比 EMA 稳"的折中方案

---

## 四、波动/通道类

### BOLL — 布林带 (Bollinger Bands)
- **看什么**:中轨(20 日 SMA)+ 上轨(+2 倍标准差)+ 下轨(-2 倍标准差)
- **怎么读**:
  - 价格触上轨 → 超买;触下轨 → 超卖
  - 带宽收窄 → 波动率下降,可能**即将突破**
  - 带宽放大 → 波动率上升,趋势启动
- **常见用法**:**震荡市**内做高抛低吸(碰上轨卖、下轨买);**突破策略**用带宽收窄作为入场前置条件

### STD — 标准差 (Standard Deviation)
- **看什么**:近 N 周期收盘价的标准差,衡量波动率
- **怎么读**:值越大 = 波动越剧烈,趋势市或事件驱动;值小 = 平静市
- **常见用法**:仓位管理(波动大仓位小)、BOLL 中轨宽度的度量基础

### RST — (HQChart 库中用法)
- **看什么**:RST 在 HQChart 的常见实现是 **"相对强度/趋势反转"类指标**(不同分支定义有差异)
- **实际含义以 HQChart 版本为准**,建议:
  - 鼠标悬停按钮看图表界面的具体解释
  - 或在容器内直接看 HQChart 源码(`wtpy/monitor/static/console/static/js/vendor.*.js` 里)
- 如果不是必需,优先用上述更通用的指标

## 推荐的看图顺序

第一次看一个回测结果,建议按这个次序叠加指标:

1. **先看 VOL** — 确认整个区间交易是否活跃,避开流动性不好的时段
2. **再看 MACD** — 快速把握主趋势方向
3. **再看 BOLL** — 判断当前处于趋势段还是震荡段
4. **根据判断选一个**:
   - 趋势段 → 加 HMA / CCI 看入场
   - 震荡段 → 加 RSI / KDJ 找反转点
5. **最后看 OBV** — 用量能确认或证伪上述结论

## 本项目策略 DualThrust 主要用哪些

从 `demos/Strategies/DualThrust.py` 源码看,这个策略的核心逻辑是:

- **观察前 N 日的最高价、最低价、收盘价** → 构造上下轨(本质是一种 BOLL 的变体)
- **突破上轨做多、突破下轨做空**
- **日内平仓**(`isForStk=False` 对期货,股票版不一定日内平)

所以理解 DualThrust 回测信号,重点看 **BOLL**(看通道)+ **VOL**(看突破是否有量)+ **MACD**(看大方向)三个。
