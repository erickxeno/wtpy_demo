# 术语表（单一真理源）

本页是整个 `docs/` 的词典。其他文档一律用 `[术语名](../00-基础/术语表.md#锚点)` 反链引用；**不内联重复解释**。

**维护规则**：

- `tag` 必填：`股票` / `期货` / `通用`，可多选。
- `首次出现` 指向第一次引入该术语的 `docs/` 文档路径；多处引用时仍只写第一次。

## 核心 20 词（最先阅读）

### CTA（商品交易顾问 / Commodity Trading Advisor）
在 WonderTrader 语境里，CTA = **基于 K 线或 Tick 的趋势/套利型自动化策略**，由 `WtEngine`（实盘）或 `WtBtEngine`（回测）驱动，策略继承自 `BaseCtaStrategy`，上下文类是 [CtaContext](./术语表.md#ctacontext).
示例：`demos/cta_stk_bt/runBT.py` 即 CTA 股票回测 demo。
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### HFT（高频交易 / High-Frequency Trading）
在本框架里指**基于委托流/逐笔成交、事件驱动**的更底层策略。调度粒度比 CTA 更细（on_tick、on_entrust、on_trans）。继承 `BaseHftStrategy`，上下文为 [HftContext](./术语表.md#hftcontext).
- tag: 期货, 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### SEL（选股 / Selection）
批量在多标的上做"择时+选股"决策的策略类型。适合多因子、多股票组合。继承 `BaseSelStrategy`，上下文为 [SelContext](./术语表.md#selcontext).
- tag: 股票
- 首次出现: docs/01-总览/wtpy是什么.md

### Bar（K 线 / Candle）
一段时间内的开高低收成交量数据。典型周期：`m1`/`m5`/`d1`。本框架用 `WtBarStruct`（C++）在内存中表示。
参考：[K 线](./术语表.md#k线).
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### Tick
逐笔行情切片：时间、价格、买卖盘口、累计成交量等。C++ 结构体 `WtTickStruct`。
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### K 线
Bar 的中文名，通常指"多根 Bar 构成的序列"。策略 `on_bar` 回调里拿到的是"最新一根"。
- tag: 通用
- 首次出现: docs/05-数据/数据流与文件格式(dsb-dmb-csv).md

### DataKit
WonderTrader 的**数据落地组件**，Python 入口 `wtpy.WtDtEngine`。职责：接收 Parser 推送的原始行情 → 按时段切分 Bar → 落盘到 `storage/csv|his|rt`。demo 入口 `demos/datakit_stk/runDT.py`。
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### Parser（行情解析器）
接入上游行情源（CTP 期货、XTP 股票、Ctp-Mini、自定义 Feed 等）的适配器。实盘里由 `tdparsers.yaml`/`mdparsers.yaml` 注入。
- tag: 通用
- 首次出现: docs/02-架构/整体架构与分层.md

### Executer（信号执行器）
把**策略目标持仓**转成**实际委托**的模块。内置算法（TWAP/VWAP/diff 等）由 `executers.yaml` 声明。
- tag: 通用
- 首次出现: docs/02-架构/整体架构与分层.md

### Trader（交易通道）
和券商/期货公司柜台对接的下单通道，如 CTP/XTP。由 `tdtraders.yaml` 配置。
- tag: 通用
- 首次出现: docs/02-架构/整体架构与分层.md

### Broker
泛指"经纪商/券商/期货公司"。Trader 是 Broker 在系统里的具体实现。
- tag: 通用
- 首次出现: docs/02-架构/整体架构与分层.md

### WtEngine
`wtpy/wtpy/WtEngine.py`，**实盘**引擎类，装配 Parser + Trader + Executer + Strategy。
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### WtBtEngine
`wtpy/wtpy/WtBtEngine.py`，**回测**引擎类，驱动历史数据回放 → 产生模拟成交 → 生成绩效报表。
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### WtDtEngine
`wtpy/wtpy/WtDtEngine.py`，**数据**引擎类（DataKit 的 Python 入口）。
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### Context（策略上下文）
策略运行时直接调用的 API 对象。CTA/HFT/SEL 各有独立 Context 类（`CtaContext`/`HftContext`/`SelContext`）。策略里 `self.__ctx__` / `context.xxx` 用法都是它。
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### Strategy（策略）
继承自 `BaseCtaStrategy` / `BaseHftStrategy` / `BaseSelStrategy` 的用户代码。实现 `on_init` / `on_bar` / `on_tick` / `on_session_begin` 等钩子。
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### 回测（Backtest）
用历史行情"重放"策略，评估其 PnL、胜率、最大回撤等指标。本框架由 `WtBtEngine` 驱动，配置入口 `configbt.yaml`。
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### 实盘（Live Trading）
用真实或模拟券商通道在交易日运行策略。由 `WtEngine` 驱动，多套 yaml 配置（`config.yaml` / `executers.yaml` / `tdparsers.yaml` / `tdtraders.yaml` / `filters.yaml` / `actpolicy.yaml`）。
- tag: 通用
- 首次出现: docs/01-总览/wtpy是什么.md

### T+1
A 股特有交易制度：**当日买入的股票，次个交易日才能卖出**。`CtaContext.stra_get_position` 在股票侧会区分"可用仓位 vs 总仓位"。
- tag: 股票
- 首次出现: docs/01-总览/wtpy是什么.md

### 滑点（Slippage）
下单价与成交价之间的偏差。回测里由 `configbt.yaml` 的 `slippage` 字段模拟；通常以"跳数"（tick 数）计。
- tag: 通用
- 首次出现: docs/06-回测/回测流程与配置(configbt.yaml).md

## 边缘/扩展词条

### subtree（git 子树）
git 的一种"把别的仓库合进来当子目录"的方式。`wtpy/` 即以 subtree 方式嵌入本仓库，便于同时追踪上游。
- tag: 通用
- 首次出现: docs/01-总览/仓库结构总览.md

### runtime（运行时）
进程启动后由加载器/解释器维护的"活体"环境。本框架的 Python runtime 负责加载 `wtpy.*` 模块 + C++ so/dll。
- tag: 通用
- 首次出现: docs/01-总览/仓库结构总览.md

### signal（信号）
策略输出的"我想要的目标仓位"（方向 + 数量）。Executer 之后才变成实际委托。
- tag: 通用
- 首次出现: docs/01-总览/仓库结构总览.md

### position（持仓）
账户当前持有的合约方向+数量。`CtaContext.stra_get_position` 可查。
- tag: 通用
- 首次出现: docs/01-总览/仓库结构总览.md

### pnl（Profit and Loss / 盈亏）
策略当前累计盈亏，分"浮动盈亏"（持仓未平）与"平仓盈亏"（已了结）。
- tag: 通用
- 首次出现: docs/01-总览/仓库结构总览.md

### 动态库（so / dll）
编译后的二进制共享库，Linux 上是 `.so`，Windows 上是 `.dll`。`wtpy/wtpy/wrapper/x64`、`linux/` 下即放这些库，由 ctypes 加载。
- tag: 通用
- 首次出现: docs/02-架构/Python与C++的边界.md

### 回调（callback）
"C++ 反过来调用 Python 函数"的技术。本框架通过 ctypes.CFUNCTYPE 注册；on_bar/on_tick 本质是 C++ 推过来的回调。
- tag: 通用
- 首次出现: docs/02-架构/Python与C++的边界.md

### ctypes
Python 标准库，用来调用 C/C++ 共享库的 FFI 工具。`wtpy/wtpy/wrapper/*.py` 都用它。
- tag: 通用
- 首次出现: docs/02-架构/Python与C++的边界.md

### singleton（单例）
同一进程内只存在一个实例的对象模式。`WtBtWrapper` / `WtWrapper` / `WtDtWrapper` 都是单例。
- tag: 通用
- 首次出现: docs/02-架构/Python与C++的边界.md

### wrapper（包装器）
对底层 C++ 动态库做一层 Python 包装，屏蔽内存布局、签名和异常转换等细节。本框架 `wtpy/wtpy/wrapper/*.py` 每个文件都是一层 wrapper。
- tag: 通用
- 首次出现: docs/02-架构/Python与C++的边界.md

### dsb / dmb
WonderTrader 的压缩二进制数据文件格式。`dsb` 一般存历史日/分钟 K 线，`dmb` 存内存映射的实时数据。通过 `WtDtHelper` 可在 csv ↔ dsb/dmb 间转换。
- tag: 通用
- 首次出现: docs/05-数据/数据流与文件格式(dsb-dmb-csv).md

### 复权（Adjusted Price）
股票价格为剔除分红/送股/拆股影响做的调整。分前复权/后复权两种。本框架在 `ContractMgr`/数据端考虑。
- tag: 股票
- 首次出现: docs/05-数据/ContractMgr与stocks.json.md

### 除权除息（Ex-right/Ex-dividend）
股票因分红送股等操作，当日开盘前对价格做除权除息处理。
- tag: 股票
- 首次出现: docs/05-数据/ContractMgr与stocks.json.md

### 停牌（Trading Halt）
股票因公司公告等原因暂停交易。策略要能识别停牌避免持仓异常。
- tag: 股票
- 首次出现: docs/05-数据/ContractMgr与stocks.json.md

### 集合竞价（Call Auction）
A 股开盘前 9:15-9:25 的撮合机制。本框架在 `SessionMgr` 的时段模板里体现。
- tag: 股票
- 首次出现: docs/05-数据/SessionMgr交易时段.md

### 一级/二级行情
一级行情：交易所直连实时；二级行情：通过数据商聚合转发，延迟更大、权限门槛低。
- tag: 通用
- 首次出现: docs/05-数据/数据流与文件格式(dsb-dmb-csv).md

### mermaid
一种用文本绘制架构/流程图的标记语言，GitHub 原生支持。本目录在 L1 文档里用它画数据流。
- tag: 通用
- 首次出现: docs/02-架构/整体架构与分层.md

### 涨跌停（Price Limit）
A 股日内涨跌幅度上限（通常 ±10%，ST 股 ±5%，创业板科创板 ±20%）。
- tag: 股票
- 首次出现: docs/10-期货补充/与股票的差异清单.md

### 保证金（Margin）
期货交易占用的资金比例。与股票的"全额买入"相对。
- tag: 期货
- 首次出现: docs/10-期货补充/与股票的差异清单.md

### 杠杆（Leverage）
期货由保证金撬动的资金倍数，放大收益也放大亏损。
- tag: 期货
- 首次出现: docs/10-期货补充/与股票的差异清单.md

### 手续费
券商/期货公司收取的交易费用。股票多为"佣金 + 印花税 + 过户费"；期货为"手数 × 固定费"或"按成交金额千分之几"。
- tag: 通用
- 首次出现: docs/10-期货补充/与股票的差异清单.md

### 印花税（Stamp Tax）
股票卖出时单边征收（目前 0.05%）。
- tag: 股票
- 首次出现: docs/10-期货补充/与股票的差异清单.md

### 换月（Roll Over）
期货主力合约从老月份滚到新月份的过程。由 `WtHotPicker` 辅助。
- tag: 期货
- 首次出现: docs/09-工具与扩展/WtHotPicker主力换月.md

### CTP
中国金融期货交易所等期货行情/交易接口协议。由中金所/上期所/大商所等使用。
- tag: 期货
- 首次出现: docs/10-期货补充/与股票的差异清单.md

### XTP
中泰证券开放股票行情/交易 API。A 股常用。
- tag: 股票
- 首次出现: docs/10-期货补充/与股票的差异清单.md

### HQChart
一套开源的 Web 端金融图表库，WtBtSnooper 用它绘制回测 K 线。
- tag: 通用
- 首次出现: docs/06-回测/WtBtSnooper可视化.md

### Sharpe / Sortino / Calmar
三种风险调整收益指标。Sharpe 用标准差，Sortino 只用下行标准差，Calmar 用最大回撤。由 `WtBtAnalyst.Calculate` 产出。
- tag: 通用
- 首次出现: docs/06-回测/WtBtAnalyst绩效分析.md

### Max Drawdown / 最大回撤
策略净值从峰值跌到谷值的最大幅度。
- tag: 通用
- 首次出现: docs/06-回测/WtBtAnalyst绩效分析.md

### 胜率（Win Rate）
盈利交易笔数 / 总交易笔数。
- tag: 通用
- 首次出现: docs/06-回测/WtBtAnalyst绩效分析.md

### 遗传算法（GA / Genetic Algorithm）
一种启发式参数优化算法，`WtCtaGAOptimizer` 用它搜索策略参数空间。
- tag: 通用
- 首次出现: docs/09-工具与扩展/CtaOptimizer参数优化.md

### tushare / rqdata / tqsdk / akshare
四个常见的国内金融数据源 Python SDK，`wtpy/apps/datahelper/` 都提供了适配。
- tag: 通用
- 首次出现: docs/09-工具与扩展/datahelper数据转换.md

### 假日表
标识非交易日（周末/法定假日）的数据文件。`SessionMgr` 依赖它决定是否推进交易日。
- tag: 通用
- 首次出现: docs/05-数据/SessionMgr交易时段.md
